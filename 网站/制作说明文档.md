# NOVA LAB 静态网站制作说明

本网站主题为“NOVA LAB 潮流数码生活馆”，定位为面向年轻用户的数码产品展示与会员服务网站。页面整体采用深色科技风格，搭配青绿色、黄色等高亮色，突出耳机、键盘、智能手表、扩展坞等产品的未来感和潮流感。网站主要包含首页、产品展示、品牌资讯、关于我们、登录、注册和找回密码等页面，导航结构清晰，用户可以在不同页面之间顺畅切换。

首页设置了自动轮播的主视觉区域，用多张产品图片展示品牌主推商品，并在“本周主推”中呈现核心产品。产品展示页按照无线音频、桌面效率、智能穿戴、生活娱乐、移动电源等分类展示商品，点击产品卡片可以弹出参数详情，包括价格、连接方式、续航、重量、适用场景等信息。品牌资讯页按新品发布、会员活动、桌面灵感和服务公告进行分类，右侧目录可跳转到对应栏目。关于我们页面介绍品牌定位、服务理念和联系方式，并加入动态数字与服务流程展示。

技术实现方面，网站使用 HTML5 搭建页面结构，CSS3 完成响应式布局、卡片样式、动画背景和移动端适配，JavaScript 实现首页轮播、产品参数弹窗、分类筛选、滚动出现、登录注册状态保存等交互功能。登录和注册功能采用 localStorage 模拟会员状态，适合静态网页作业展示。网站已部署到 GitHub Pages，可通过浏览器在线访问。
