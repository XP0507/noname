# “NOVA LAB 潮流数码生活馆”静态网站制作说明

本次网页前端设计作业以“NOVA LAB 潮流数码生活馆”为主题，定位为面向年轻用户的数码产品展示与会员服务网站。网站使用 HTML5 与 CSS3 独立完成，文件结构包括首页 index.html、登录页 login.html、注册页 register.html、产品展示页 destinations.html、品牌资讯页 news.html、关于我们页 about.html、找回密码页 forgot.html、统一样式文件 css/style.css，以及 assets 文件夹中的本地产品视觉素材。

网站首页采用完整的盒子模型布局，顶部包含品牌 Logo、网站名称、公告栏、联系电话和主导航菜单；主导航可以跳转到首页、产品展示、品牌资讯、关于我们、登录和注册会员等页面。首页主体突出品牌视觉，设置了产品展示型 Hero 区域、核心数据展示、本周主推产品、会员活动和资讯列表。页脚统一放置版权信息、联系方式、地址、邮箱和备案号，符合课程要求中的网站标题、主导航、内容展示区、信息分区、版权与联系方式等评分点。

登录页和注册页均使用 HTML 表单完成。登录页包含账号、密码、保持登录、忘记密码和注册入口；注册页包含用户名、手机号、邮箱、密码、感兴趣品类和服务条款勾选项。表单使用 required、type="email"、type="tel"、pattern、minlength 等 HTML5 验证属性，能够完成基本的非空、邮箱格式、手机号格式、用户名格式和密码长度验证。输入框、按钮、标签和提示文字统一由 CSS 控制，页面整体风格与首页保持一致。

其他页面方面，产品展示页用于展示无线耳机、机械键盘、智能手表、无线音箱等数码产品，并列出产品特点；品牌资讯页用于发布新品预约、桌面方案、会员售后等信息；关于我们页介绍品牌定位、网站功能和联系方式。所有页面之间通过导航栏相互链接，且共用同一个 CSS 文件，保持统一的配色方案、字体规范、卡片样式、按钮样式和页头页脚。

整体视觉采用深色科技背景，搭配青绿色、亮黄色、蓝色和珊瑚红作为重点色，突出数码品牌的年轻感和现代感。布局上使用 Flex 和 Grid 完成响应式排版，使电脑端和移动端都能正常浏览。相比普通模板页面，本网站更强调产品视觉、品牌氛围和信息层级，既满足静态网站作业的功能要求，也具有更强的展示效果。
