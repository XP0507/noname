# “海丝北海”静态网站制作说明

本次网页前端设计作业以“海丝北海城市文旅网站”为主题，面向来北海旅游的游客提供景点展示、新闻公告、会员登录注册和平台介绍等信息服务。网站采用 HTML5 与 CSS3 完成，所有页面均为静态页面，文件结构清晰，主要包括首页 index.html、登录页 login.html、注册页 register.html、景点展示页 destinations.html、新闻动态页 news.html、关于我们页 about.html、找回密码页 forgot.html、统一样式文件 css/style.css，以及 assets 文件夹中的页面图片素材。

网站首页重点体现完整的盒子模型布局。页面顶部包含 Logo、网站标题、联系电话和主导航菜单；导航菜单能够跳转到首页、景点展示、新闻动态、关于我们、登录和注册等页面。首页主体设置了 Banner 轮播区域，通过 CSS 动画实现多张城市宣传图自动切换，内容展示区包括城市速览、热门景点、旅行公告和服务分区。页脚统一放置版权信息、联系方式和备案号，符合题目对版权、联系方式、备案号等信息分区的要求。

登录页和注册页都使用 HTML 表单制作。登录页包含账号、密码、记住我、忘记密码和注册链接；注册页包含用户名、手机号、邮箱、密码、旅行偏好和服务条款勾选项。表单使用 required、type="email"、type="tel"、pattern、minlength 等 HTML5 验证属性，能够完成基本的非空、邮箱格式、手机号格式和密码长度检查。页面使用统一的表单样式，输入框、按钮、间距和背景图保持一致，视觉效果较为整洁。

其他页面方面，景点展示页用于展示北海银滩、涠洲岛、百年老街、海丝文化港等旅游产品；新闻动态页用于展示景区公告、交通提醒和活动资讯；关于我们页说明网站定位，并提供联系电话、邮箱和地址。各页面之间通过导航栏互相链接，所有页面共用同一个 CSS 文件，保持了统一的配色、字体、按钮、卡片、页头和页脚规范。

整体设计采用青绿色、橙色和浅米色作为主要配色，既体现滨海城市的清爽感，也避免页面过于单调。布局上使用 Flex 和 Grid 完成响应式排版，在电脑和较窄屏幕下都能正常浏览。网站主题明确、页面数量充足、结构完整，能够满足课程考核中关于首页、登录页、注册页、其他页面、全站导航、统一 CSS 样式和制作说明文档的要求。
